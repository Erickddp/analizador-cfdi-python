# Analizador CFDI — Sitio Web

Este directorio contiene el código fuente de la landing page para la demo técnica del **Analizador CFDI**.

## Estructura
- `/index.html`: Estructura principal.
- `/styles.css`: Estilos (Tema claro/oscuro minimalista).
- `/script.js`: Lógica de UI (Carrusel, Toggle de tema).
- `/assets`: Recursos estáticos (Imágenes, Descargas).

## Despliegue
Este sitio está configurado para desplegarse mediante **GitHub Pages** desde la rama principal o la carpeta `/site`.
